1. 개인 프로젝트

이 프로젝트는 메인 페이지를 바탕으로 구현한 웹 페이지로, 메인 페이지에서 사이트의 주요 기능을 소개하며 회원가입, 로그인, 프로필 관리 기능을 포함한 프로젝트입니다.

2. 개인 프로젝트

이 프로젝트는 메인 페이지와 함께 뉴스, 영화, 정보 관련 페이지를 구현한 웹 페이지로, 각 페이지는 해당 주제에 맞는 콘텐츠를 제공합니다. 이 프로젝트는 반응형 디자인을 적용하여 다양한 화면 크기에서 최적화된 사용자 경험을 제공한 프로젝트입니다.

3. 개인 프로젝트

SKILL :

4. 팀별 프로젝트

이 프로젝트는 애플 사이트의 메인 페이지를 참고해 웹 페이지를 디자인하고, 반응형 웹 디자인을 적용해 다양한 기기에서 최적화된 경험을 제공하는 작업입니다. HTML, CSS, JavaScript등을 사용해 레이아웃과 기능을 구현한 프로젝트입니다.

SKILL :

5. 포토샵 프로젝트

이 프로젝트는 포토샵과 피그마를 활용하여 이벤트에 관한 배너를 자유롭게 제작한 작업입니다. 제작한 배너는 정부부처, 은행, 대기업, 자유 소재와 카드뉴스, 명함 소재를 반영하여 디자인한 프로젝트 입니다.
